Dwell Time เฉลี่ยรายเดือน (Normal Segment) — Import vs Export
คำนวณเฉพาะ Normal segment (2-30 วัน) ตามมาตรฐาน skill · เส้นทึบ = mean · เส้นประ = median · แถบเหลือง = ช่วงล่าสุด FY2568-69
ประเด็นสำคัญ
Import dwell time เฉลี่ยขยับจาก 5.20 วัน → 5.80 วัน (+11.5%) ส่วน Export ขยับจาก 5.51 วัน → 6.15 วัน (+11.6%) ทั้งสองทิศทางใกล้เคียงกันมาก บ่งชี้ว่าน่าจะเป็นปัจจัยระดับท่าเรือ (เช่น ความหนาแน่นลาน หรือกระบวนการผ่านพิธีการ) มากกว่าปัจจัยเฉพาะกลุ่มสินค้าใดกลุ่มหนึ่ง — ควรสอบถามทีม yard operation เพื่อยืนยันสาเหตุ
Import ปลอดภัย ≤5.5d / Export ≤7.0d Import ⚠️ 5.5-6.5d / Export ⚠️ 7.0-8.0d Import 🔴 >6.5d / Export 🔴 >8.0d
Segmentation รายเดือน — Express / Normal / Anomaly (%)
% ของตู้ในแต่ละชั้น ต่อเดือน — Anomaly ที่สูงผิดปกติควรเช็ค batch artifact ก่อนสรุปว่าเป็นปัญหาจริง
ประเด็นสำคัญ
ตู้ในกลุ่ม Anomaly (ค้างเกิน 30 วัน) ยังเป็นสัดส่วนน้อยทุกเดือน (ต่ำกว่า 2.5%) แต่มีทิศทางสูงขึ้นต่อเนื่องในปีล่าสุดเทียบ baseline — เป็นกลุ่มความเสี่ยงสูงสุดต่อค่าใช้จ่าย demurrage/storage ถึงปริมาณจะน้อยแต่ควรติดตามแยกจากแนวโน้มหลัก
เทียบ Baseline vs ช่วงล่าสุด (Normal Segment)
Baseline = ก.ค.2565-ธ.ค.2567 · ล่าสุด = ม.ค.2568-เม.ย.2569
ประเด็นสำคัญ
ทั้ง mean และ median ขยับขึ้นเล็กน้อยทุกตัว — ไม่ใช่การเปลี่ยนแปลงรุนแรง แต่สอดคล้องกันทั้ง Import/Export จึงน่าเชื่อว่าเป็นแนวโน้มจริง ไม่ใช่ความผันผวนรายเดือน
Warehouse Intensity (% dwell > 7 วัน)
รวมทุก segment ที่ dwell>7 วัน — ตัวขับเคลื่อนรายได้ Storage และสัญญาณ "คลังกลางกรุงเทพ"
ประเด็นสำคัญ
% ตู้ค้างเกิน 7 วันคือจุดที่ค่าธรรมเนียมฝากตู้ขั้นบันไดเริ่มขยับ — ดูความเชื่อมโยงกับรายได้จริงในหัวข้อ "เชื่อมโยงการเงิน" ด้านล่าง
เทียบฐานรายปีงบประมาณ — Dwell Time เฉลี่ย (Normal Segment)
ปีงบประมาณ ต.ค.-ก.ย. (เลขปี พ.ศ. = ปีที่สิ้นสุด ก.ย.) · ตามรอบที่ฝ่ายการเงินใช้ติดตามรายได้
ประเด็นสำคัญ
ฐาน FY2566-2567 เฉลี่ยรวม: Import 5.18 วัน / Export 5.37 วัน — FY2568 ขยับเป็น Import 5.70 วัน / Export 6.28 วัน และ FY2569 (มีข้อมูลถึง เม.ย.69 เท่านั้น) อยู่ที่ Import 5.72 วัน / Export 5.92 วัน ทิศทางขึ้นต่อเนื่อง 2 ปีติด ไม่ใช่ความผันผวนปีเดียว — อ่านวิธีนี้ง่ายกว่า: "ปีฐานเป็นแบบนี้ ตอนนี้เป็นแบบนี้ ต่างกัน X วัน" แทนการดูกราฟ mean/median รายเดือนที่ละเอียดเกินไป
* FY2565 มีข้อมูลแค่ ก.ค.-ก.ย.65 (3 เดือน) และ FY2569 มีข้อมูลถึง เม.ย.69 (7 เดือน) — เป็นค่าเฉลี่ยจากเดือนที่มีข้อมูลจริงเท่านั้น ไม่ใช่ค่าทั้งปี
กระจายตัวจำนวนวันที่ตู้อยู่ในลาน — Baseline vs Recent
% ของตู้ในแต่ละช่วงวัน (bucket) เทียบ Baseline (ก.ค.2565-ธ.ค.2567) กับ Recent (ม.ค.2568-เม.ย.2569)
ประเด็นสำคัญ
ตู้ Import ส่วนใหญ่อยู่ในลาน 2-3 วัน (34.5% baseline → 31.0% recent) ตู้ Export ส่วนใหญ่อยู่ 4-5 วัน (37.8% baseline → 33.2% recent) — พฤติกรรมหลักไม่เปลี่ยน แต่ "หาง" ของกระจายตัวขยายออก: สัดส่วนตู้ค้างเกิน 30 วันเพิ่มขึ้นเกือบ 3 เท่าทั้ง Import (0.4%→1.4%) และ Export (0.3%→0.6%) แม้ฐานยังเล็ก นี่คือสัญญาณเริ่มต้นที่ควรติดตาม ไม่ใช่ตัวเลขที่น่าตกใจในตัวเอง
ผลกระทบต่อรายได้จากตู้ค้างเพิ่มขึ้น — เทียบฐาน FY2566-2567
คำนวณจากอัตราค่าภาระฝากตู้จริง (หนังสือค่าภาระ ท่าเรือกรุงเทพ พ.ศ.2543 ตู้มีสินค้า FCL) × ขนาดตู้จริงของตู้แต่ละใบ (nominal_length: 20/40/45 ฟุต จากไฟล์ raw) คูณกับจำนวนตู้จริงแต่ละปี
ประเด็นสำคัญ
ค่าธรรมเนียมเฉลี่ยต่อตู้ (Export) พุ่งขึ้น +76.5% ในปี FY2569 ทั้งที่ dwell time เฉลี่ยขยับขึ้นแค่ +0.55 วัน — เพราะโครงสร้างค่าภาระเป็นขั้นบันได ตู้ส่วนน้อยที่ไหลเข้ากลุ่มค้างนาน (tail ที่ขยายตัวเกือบ 3 เท่าจากหัวข้อ "กระจายตัว" ด้านบน) ดึงค่าเฉลี่ยรายได้ต่อตู้ขึ้นแรงกว่าที่ตัวเลข dwell เฉลี่ยอย่างเดียวจะบอกได้

เรื่องขนาดตู้: ตู้ Import มีสัดส่วนค้างที่ 20ฟุต:40ฟุต ≈ 53:47 และ Export ≈ 58:42 มาตลอดทุกปี (ไม่เปลี่ยนแปลงมาก) แปลว่าผลกระทบรายได้ที่เห็นนี้มาจากตู้ค้างนานขึ้นจริง ไม่ใช่เพราะ mix ขนาดตู้เปลี่ยน — ตัวเลขรวมยังเป็นค่าประมาณจากอัตราปี 2543 ไม่ใช่รายได้จริงที่บันทึกบัญชี เพราะยังไม่มีเล่มอัตราฉบับปรับปรุงล่าสุดมาเทียบ
เชื่อมโยงการเงิน — % ตู้ค้างเกิน 7 วัน vs รายได้ค่าภาระฝากตู้สินค้า
รายได้จริงจาก Power BI_8 Months_69.xlsx (ทกท., ทุกบรรทัดที่มีคำว่า "ฝากตู้") เทียบ % ตู้ dwell>7 วันรายเดือน ต.ค.2565-เม.ย.2569
ประเด็นสำคัญ — ตรงไปตรงมา
Correlation ระหว่าง % ตู้ค้างเกิน 7 วัน กับรายได้ค่าฝากตู้ = 0.34 — ทิศทางสอดคล้องกันตามสมมติฐาน "Chain A" ใน skill doc แต่เป็นความสัมพันธ์ระดับปานกลาง ไม่ใช่ตัวขับเคลื่อนหลักเดี่ยวๆ เพราะรายได้ค่าฝากตู้ยังขึ้นกับปัจจัยอื่นร่วมด้วย (mix FCL/LCL, ปริมาณตู้รวม, ส่วนลด/อัตราค่าธรรมเนียมที่ปรับเปลี่ยนรายงวด) ส่วน % ตู้ค้างเกิน 30 วัน มี correlation แค่ -0.06 กับรายได้ — แทบไม่มีความสัมพันธ์ เพราะปริมาณยังน้อยเกินกว่าจะขยับรายได้รวมได้อย่างมีนัยสำคัญ

คำแนะนำการใช้: เหมาะเสนอเป็น "สัญญาณทิศทางสอดคล้องกัน สนับสนุนสมมติฐาน Chain A" ไม่เหมาะเสนอเป็น "หลักฐานว่า dwell time เป็นตัวขับเคลื่อนรายได้หลัก" เพราะจะเกินกว่าที่ข้อมูลสนับสนุนจริง

📌
Batch artifact ฝั่ง Export อ่านเป็น "จำนวนตู้ผิดปกติ" ตรงๆไม่ได้ — ตรวจพบว่า timestamp time_out ของฝั่ง Export ถูกบันทึกที่ความละเอียดหลักนาที (99.7% ตรงกับช่วง 5 นาทีเป๊ะ) ขณะที่ฝั่ง Import บันทึกละเอียดถึงระดับวินาที (มีแค่ 1.7% ที่ตรงวินาที :00) — ทำให้เกณฑ์เดิม "ตู้เกิน 5 ใบ ที่ time_out ตรงกันเป๊ะ = batch artifact" ติดเกือบทุกกลุ่มของฝั่ง Export โดยธรรมชาติ ไม่ใช่สัญญาณความผิดปกติจริง (น่าจะมาจากการบันทึกเวลายกตู้ขึ้นเรือเป็นช่วงเช็คพอยท์ ไม่ใช่ real-time) ตัวเลข exp_batch_n ในกราฟนี้จึงเป็นค่าอ้างอิงคร่าวๆเท่านั้น ไม่กระทบความถูกต้องของค่า dwell แต่ละตู้ หรือ mean/median ของ Normal segment เพราะ dwell คำนวณจาก time_in→time_out ของตู้นั้นจริง ไม่ใช่จากกลุ่ม batch
